# Basic Network Sniffer
### CodeAlpha Cybersecurity Internship — Task 1

A Python-based network packet sniffer that captures and analyzes 
real network traffic in real time using raw sockets.

## Features
- Captures TCP, UDP and ICMP packets
- Shows source/destination IP and ports
- Identifies services (HTTP, HTTPS, DNS, SSH, FTP etc.)
- Displays payload in HEX and ASCII format
- Saves full session log to JSON

## How to Run
Run PowerShell as Administrator then:
python network_sniffer.py

## Requirements
- Python 3.x
- No external libraries needed
- Must run as Administrator (Windows) or sudo (Linux)

## Output
- Live packet display in terminal
- sniffer_log.json saved after session
